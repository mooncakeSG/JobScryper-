# Reports Directory

This directory contains exported ATS analysis and bias detection reports generated by the Auto Applier system.

## Report Types

### ATS Analysis Reports
- **Format**: JSON and TXT files
- **Content**: 
  - Resume-job match scores
  - Missing keywords analysis
  - ATS optimization recommendations
  - Keyword importance rankings

### Bias Detection Reports
- **Format**: JSON and TXT files
- **Content**:
  - Bias pattern detection results
  - Inclusive language analysis
  - Red flag identification
  - Diversity and inclusion recommendations

## File Naming Convention

Reports are automatically named using the following pattern:
- `ats_report_{job_title}_{company}_{timestamp}.json`
- `ats_report_{job_title}_{company}_{timestamp}.txt`
- `bias_analysis_{job_title}_{company}_{timestamp}.json`

Example:
```
ats_report_IT_Support_Specialist_TechCorp_20250107_115830.json
ats_report_IT_Support_Specialist_TechCorp_20250107_115830.txt
```

## Report Usage

### Downloading Reports
- Use the "Download ATS Report" button in the Streamlit app
- Reports are automatically generated for each job match analysis
- Both JSON (machine-readable) and TXT (human-readable) formats available

### Report Analysis
1. **ATS Score**: 0-100% match score between resume and job requirements
2. **Missing Keywords**: Critical skills/terms absent from resume
3. **Bias Flags**: Potentially problematic language in job postings
4. **Recommendations**: Actionable advice for improving match scores

### Integration
- Reports can be imported into spreadsheet applications
- JSON format suitable for data analysis and tracking
- TXT format ideal for quick review and sharing

## Cleanup
Reports are retained indefinitely for historical analysis. Consider periodic cleanup of older reports to manage disk space.

## Privacy Note
Reports contain personal resume information and should be handled according to your organization's data privacy policies.
